# internet-image-replacer3000-
This is a chrome extension,
in order to use it, you'll need:
  - To first clone the "Image_replacing_internet-extension" file on your side
  - Open chrome
  - Go to "chrome://extensions"
  - Click "Load unpacked"
  - Select the file
  - Click on the extension button (puzzle piece at the top right)
  - Select Flavor Replacer
  - Type whatever you want and click "save"
