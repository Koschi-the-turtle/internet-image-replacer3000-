# internet-image-replacer3000-
This is a chrome extension,
in order to use it, you'll need to download the zip file and unzip it, Open chrome,
type chrome://extensions in the address bar (this will get you to the extension menu)
at the top left corner of the page you'll find the "Load unpacked" button and you need ton click on it
and select the unziped file. Then go on whatever website and click on the extension button (the puzzle piece right next to the address bar),
and choose the Flavor Replacer extension, which will open the popup in which you'll be able to enter the tag of your choice before clicking save to apply the changes.
